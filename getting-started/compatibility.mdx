# Compatibility

<Note>
  **Before Injection**: Please verify your Windows version matches the supported list below.
</Note>

## Compatibility Setup

Check which version of Windows 10 or 11 you are running.

<Steps>
  <Step title="Run Command">
    Enter **winver** in the Windows search bar and execute the resulting command.
  </Step>
  <Step title="Verify Version">
    Take note of the version displayed (e.g., **22H2**, **23H2**).
  </Step>
</Steps>

### OS Support Table

<Warning>
  If a box is crossed out, that specific Windows version is **not supported** for that game. Attempting to inject may result in system instability.
</Warning>

### Incompatible Windows Version?

If your specific Windows version is not supported, you must either reinstall your operating system or install a compatible version on a separate drive.
