# System Preparation

Before downloading or running the loader, you must prepare your environment to prevent crashes, or detection issues.

<Steps>
  <Step title="Disable Windows Defender">
    Windows Defender must be **fully disabled**, not just paused. We recommend using the **Defender Control** tool to ensure it does not turn itself back on.

    <Card title="Download Defender Control" icon="shield-slash" href="https://www.sordum.org/9480/defender-control-v2-1/">
      Click here to download the tool required to permanently disable Windows Defender.
    </Card>

    <Note>
      **Third-Party Antivirus:**

      - **Kaspersky:** Can usually be enabled or temporarily disabled.
      - **Bitdefender / Norton / Avast:** Must be **fully uninstalled**.
    </Note>
  </Step>
  <Step title="Uninstall Third-Party Anti-Cheats">
    Conflicting anti-cheat drivers will prevent the loader from initializing. You must uninstall the following:

    - **Riot Vanguard** (Valorant)
    - **FaceIT** Anti-Cheat
    - **ESEA** Anti-Cheat
  </Step>
  <Step title="Remove Conflicting Software">
    The following applications may conflict with our cheat. Please uninstall or disable them:

    - **Software:** Cheat Engine, Network Software (e.g., Fiddler, Wireshark)
    - **Overlays:** Overwolf
    - **Bloatware:** ASUS Armoury Crate, MSI Center/Dragon Center, Razer Synapse
    - **Wallpapers:** Wallpaper Engine
  </Step>
  <Step title="Hardware Consistency (HWID)">
    The loader binds to your hardware configuration the moment you log in.
  </Step>
</Steps>
