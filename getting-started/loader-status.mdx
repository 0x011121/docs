## Understanding Status Labels

The status page primarily indicates if a cheat is **Undetected**, **Updating**, or **Detected**.

<Note>
  **Important:** The status page may not immediately reflect **game updates**, small **testing phases**, or temporary **maintenance**. For the most up-to-the-minute information, please check our Discord.
</Note>

When you open the loader, you will see one of the following tags next to your cheat:

### 🟢 LOAD

**The cheat is undetected and ready to use.** This status means the software is fully functional and considered safe for use.

### 🟡 TESTING

**Use at your own risk.** The cheat is currently being tested. This usually occurs after a game update, during bug hunting, or when verifying detection status.

<Warning>
  We do not recommend using the cheat while it is in **TESTING** mode.
</Warning>

### 🔴 DOWN

**The cheat is offline.** The software has been temporarily disabled. This typically happens immediately after a game update while we work on a new version. You will not be able to inject the cheat while it is in this state.
