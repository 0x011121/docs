# Menu Not Interactive

If the cheat menu is visible on your screen but you are **unable to click, interact, or navigate** the menu options, the issue is typically caused by conflicting software or display scaling settings.

## Interaction and Display Fixes

<Steps>
  <Step title="Match Resolutions">
    Ensure your **in-game resolution** is set to match your **desktop resolution** exactly. Any mismatch can cause input misalignments with the menu.
  </Step>
  <Step title="Check Windows Scaling (100%)">
    The cheat menu is designed for 100% display scaling. If you are using anything higher (e.g., 125% or 150%), the menu's hitbox may be offset.

    1. Right-click your desktop and select **Display Settings**.
    2. Under **Scale and layout**, ensure the percentage is set to **100%**.
  </Step>
  <Step title="Disable Steam Overlay Scaling">
    The Steam overlay feature can conflict by scaling its own text and icons, affecting the cheat menu's mouse input.

    1. Open **Steam Settings**.
    2. Navigate to the **In-Game** section.
    3. **Disable** the option: "Scale Steam Overlay text and icons to match monitor settings."
  </Step>
  <Step title="Ensure Steam Overlay is Enabled">
    The menu often relies on the game client's overlay for input processing.

    - Make sure the general **Steam Overlay** is enabled within the Steam client settings.
  </Step>
  <Step title="Disable Discord Presence (DayZ Only)">
    If you are a DayZ subscriber, the game launcher's Discord integration can cause issues.

    - Open the **DayZ SA Launcher settings** and disable **"Discord Presence."**
  </Step>
  <Step title="Remove Conflicting Overlays and Software">
    Software that draws an overlay or hooks into the graphics pipeline often conflicts with the menu's input.

    - **Uninstall** or fully close software like: **Overwolf**, **Wallpaper Engine**, **Razer Synapse**, **MSI bloatware**, and **ASUS Armoury bloatware**.
  </Step>
  <Step title="Check Steam Beta Status">
    If you are currently enrolled in the **Steam Beta** program, this can sometimes introduce stability issues.

    - Go back to the **normal, non-beta version** of the Steam client.
  </Step>
  <Step title="Final Restart">
    After making any of the changes above, **restart your PC** completely to ensure all system settings and services are properly reset.
  </Step>
</Steps>
