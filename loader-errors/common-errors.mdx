# Code 869, 154, ...

If you receive any of the following error codes upon attempting to inject, it signifies a conflict with security software running on your PC:

**Code 869, Code 154, Code 2310, Code 28, Code 34, Code 67**

## Cause

<Warning>
  **Kernel-Level Driver Conflict**

  These errors are caused by **competitor anti-cheats** (like Vanguard or FACEIT) or **third-party antivirus programs** that use kernel-level drivers. These drivers interfere with the injection process and must be removed.
</Warning>

## Solution

You must ensure that all third-party security software is fully uninstalled and that Windows Defender is disabled.

<Steps>
  <Step title="Uninstall Competitor Anti-Cheats and AVs">
    You must **fully uninstall** the following software from your system, as simply disabling them will not work:

    - **Riot Vanguard** (Valorant)
    - **FACEIT** Anti-Cheat
    - **Any third-party antivirus** (e.g., Bitdefender, Norton, McAfee)
  </Step>
  <Step title="Fully Disable Windows Defender">
    Windows Defender must be completely disabled. Pausing it is not sufficient, as it will often re-enable itself during the injection process.

    <Note>
      We strongly recommend using the **Defender Control** tool to ensure the service remains off.
    </Note>
  </Step>
</Steps>
