# Not Authorized

If you receive the error **"You are not authorized to use this account"**, it means your current **Hardware ID (HWID)** does not match the unique ID recorded when you first successfully logged into the loader.

<Note>
  **First-Time Usage**

  Even if this is the very first time you are using our product, this error can appear due to an initial issue. The fix requires opening a support ticket.
</Note>

## HWID Troubleshooting

Your HWID is a unique key derived from your PC's components (CPU, Motherboard, storage devices, etc.). The loader is tied to this key.

<Steps>
  <Step title="Check for Hardware Changes">
    Think back to when you last successfully used the loader. Did you make any changes since then?

    - **Monitors:** Did you disable, enable, or swap a monitor?
    - **Storage:** Did you add or remove an SSD or HDD?
    - **USB Devices:** Did you add or remove any permanent USB drive?
    - **VPN:** Did you enable a VPN?

    Even minor changes can alter your HWID.
  </Step>
  <Step title="Open a HWID Reset Ticket">
    If you did make any changes to your PC's hardware configuration, your HWID is different. You will need to request a reset.

    <Card title="Open HWID Reset Ticket" icon="sync" href="https://proofcore.io/forums/support/">
      Click here to open a support ticket specifically requesting a HWID reset.
    </Card>
  </Step>
  <Step title="Verify Loader Download Source">
    If someone sent you the loader executable directly (e.g., via Discord or email), it might be corrupted or outdated.

    Please delete the file you have and download it again while logged into your own account from the official file section:

    <Card title="Official Loader Download" icon="download" href="https://proofcore.io/forums/files/">
      Download the clean, official loader file from your account here.
    </Card>
  </Step>
  <Step title="Contact Support">
    If you are absolutely certain that your PC hardware is exactly the same as when you last used the loader, please open a general support ticket.

    <Card title="Open Support Ticket" icon="headset" href="https://proofcore.io/forums/support/">
      Contact support here, and let them know you have already verified your hardware configuration.
    </Card>
  </Step>
</Steps>
