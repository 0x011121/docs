- Set your game display mode to **Borderless** (recommended) or **Windowed Mode**. **Fullscreen mode will not work!**

## How to Inject

Follow these steps precisely for a successful injection:

<Steps>
  <Step title="Prepare Steam">
    Run Steam. **Crucially**, ensure the Steam overlay is **enabled** and that you are **not** using the Steam Beta client version.
  </Step>
  <Step title="Load the Cheat">
    1. Run the loader as **administrator**.
    2. Log in using your website credentials (email and password).
    3. Select **DayZ** and press **Load**.
  </Step>
  <Step title="Launch Game">
    Wait for the loader to close itself. You **must run DayZ** within **2 minutes** after the loader has closed.
  </Step>
  <Step title="Open Menu">
    Once you are in-game, open the menu by pressing the **Insert** key.
  </Step>
</Steps>

<Note>
  It can take up to 5 minutes for the menu to show up. If it still doesn't show up, the injection did not succeed or you haven't prepared your computer or game correctly. Refer back to the **System Preparation** guide.
</Note>
