## How to Inject

Follow these steps precisely for a successful injection:

<Steps>
  <Step title="Prepare Steam">
    Run Steam. **Crucially**, ensure the Steam overlay is **enabled** and that you are **not** using the Steam Beta client version.
  </Step>
  <Step title="Loader Injection">
    1. Run the loader as **administrator**.
    2. For first-time users register an account using your product key along with your    preferred username and password.
    3. Log in using the username and password you just created during registration.
    4. There is no in-game menu available, so all settings must be configured before game launch or by using one of our provided configs.
    5. press **Inject**.
  </Step>
  <Step title="Launch Game">
    Wait for the loader to close itself. You **must run PUBG** within **3 minutes** after the loader has closed.
  </Step>
</Steps>

<Note>
  If ESP does not appear in the training range, carefully review all previous steps. If everything is configured correctly, reboot your system and perform the injection process again.
</Note>
