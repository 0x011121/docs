# Call of Duty

This guide outlines the necessary system requirements and the injection process for the Call of Duty cheat.

## Before You Inject

### System Requirements

Ensure your system meets the following requirements to prevent issues:

<CardGroup cols={2}>
  <Card title="System Preparation" icon="wrench" href="/getting-started/preparation">
    Disable overlays and run the loader as administrator.
  </Card>

  <Card title="Compatibility" icon="info-circle" href="/getting-started/compatibility">
    Windows 10 and Windows 11 are supported.
  </Card>
</CardGroup>

### Supported Launchers

The cheat is compatible with the following game clients:

<CardGroup cols={3}>
  <Card title="Battle.net" icon="desktop">
    Launch via the Battle.net client.
  </Card>

  <Card title="Steam" icon="steam-symbol">
    Launch via the Steam client.
  </Card>

  <Card title="Xbox Gamepass" icon="xbox">
    Launch via the Xbox app.
  </Card>
</CardGroup>

### Game Preparation

- Set your game display mode to **Borderless** (recommended) or **Windowed Mode**. **Fullscreen mode will not work!**

## How to Inject

Follow these steps precisely for a successful injection:

<Steps>
  <Step title="Prepare Launcher">
    Run the desired game launcher.
  </Step>
  <Step title="Load the Cheat">
    1. Run the loader as **administrator**.
    2. Log in using your website credentials (email and password).
    3. Select **Call of Duty** and press **Load**.
  </Step>
  <Step title="Launch Game">
    Wait for the loader to close itself. You **must run Call of Duty** within **2 minutes** after the loader has closed.
  </Step>
  <Step title="Open Menu">
    Once you are in-game, open the menu by pressing the **Insert** key.
  </Step>
</Steps>

<Note>
  It can take up to 5 minutes for the menu to show up. If it still doesn't show up, the injection did not succeed or you haven't prepared your computer or game correctly. Refer back to the **System Preparation** guide.
</Note>
